# Responsive Student Registration Form

## 📖 About the Project

This is a responsive Student Registration Form developed using **HTML**, **CSS**, and **JavaScript**. The project allows users to enter student details through a form. After submitting the form, the entered information is displayed dynamically in a table without refreshing the page.

The project is designed to demonstrate the fundamentals of front-end web development, including form handling, DOM manipulation, responsive design, and JavaScript event handling.

---

## 🚀 Features

* Student Registration Form
* Required field validation
* Gender selection using radio buttons
* Dynamic table row creation
* Prevents page refresh using JavaScript
* Automatically clears the form after submission
* Responsive design for desktop, tablet, and mobile
* Clean and beginner-friendly code

## 📌 How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
4. Fill in the student details.
5. Click the **Submit** button.
6. The submitted details will be displayed in the table below the form.

---

## 📸 Project Features

* Responsive user interface
* Form validation using the `required` attribute
* JavaScript `addEventListener()` for form submission
* Dynamic DOM manipulation using `createElement()` and `appendChild()`
* Automatic form reset after successful submission
